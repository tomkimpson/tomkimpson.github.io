Code

In the spirit of reproducible research (see also Trisovic et al. 2022) I try, where possible, to publicly share the code associated with my published papers.

  • RelativisticDynamics.jl Differentiable, number format flexible, spin curvature dynamics.

  • SpeedyWeather.jl Global atmospheric modelling for climate physics. Minor contributions.

  • ML4L. Machine learning for land surface modeling.

  • Iliad. What is the timing signal from a pulsar in a strong-field relativistic environment?

  • Quadrupole Orbital Dynamics. More spin-curvature dynamics, but this time for a metric with a arbitrary mass quadrupole.

  • RayTracing. In a Kerr spacetime

  • GW_bursts. Numerical kludge gravitational waveforms and SNRs for pulsar in extreme mass ratio systems.

Please see github for a list of all public repositories.
